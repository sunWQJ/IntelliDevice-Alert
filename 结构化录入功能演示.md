# 🏗️ 结构化录入功能演示

## 功能概述

新实现的结构化录入功能将传统的表单录入转变为智能化的结构化分析流程，主要包含以下特性：

### 🎯 核心功能

1. **智能字段提示** - 每个输入框都有清晰的标签和示例
2. **结构化分析** - 自动从自然语言中提取关键医疗信息
3. **标准术语匹配** - 将提取的信息与医疗标准术语库匹配
4. **置信度评估** - 提供分析结果的可信度评估
5. **LLM重构优化** - 支持使用大模型重新优化描述
6. **可视化确认** - 结构化结果预览和确认界面

### 📋 界面改进

#### 1. 分组化信息录入
- **基本信息**（蓝色背景）：医院、设备、制造商、型号等
- **事件描述**（黄色背景）：详细的情况说明，包含示例提示
- **伤害评估**（红色背景）：严重程度和处置措施

#### 2. 智能化按钮
- 📝 **直接提交** - 传统方式直接录入
- 🔍 **提交并标准化** - 提交同时进行术语标准化
- 🏗️ **结构化分析** - 新的智能分析流程
